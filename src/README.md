## 👋 欢迎来到 Lin's Wiki

这是一个专注于**工程造价**领域的个人知识库，旨在整理和分享工程造价相关的专业知识。

## 📚 知识体系

### 工程计量
- 土建计量：建筑面积、土方、混凝土、砌筑、装饰
- 钢筋计量：钢筋计算规则、翻样技巧

### 工程计价
- 预算编制：预算编制方法、招投标报价
- 清单计价：清单计价规范
- 定额套用：定额体系、套用方法
- 造价分析：造价分析、审核、成本控制

### 法律法规
- 现行法律：宪法、民法商法、行政法、经济法等
- 行政法规：国务院制定的行政法规
- 部门规章：国务院各部门制定的规章
- 工程建筑规范：国家标准、行业规范
