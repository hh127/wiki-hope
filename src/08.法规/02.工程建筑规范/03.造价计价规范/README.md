# 造价计价规范

本目录收录工程造价计价相关的国家标准、地方定额及地区单位估价表。

## 目录结构

### 01.国家计价规范
国家标准（GB）中的计价规范和工程量计算规范：
- 《建设工程工程量清单计价规范》（GB50500-2013）
- 《房屋建筑与装饰工程工程量计算规范》（GB50854-2013）
- 《通用安装工程工程量计算规范》（GB50586-2013）

### 02.消耗量定额
#### 新疆
- 房屋建筑与装饰工程消耗量定额（2020）
- 市政工程消耗量定额（2020）
- 安装工程补充消耗量定额（2020）
- 园林绿化工程消耗量定额（2014）

### 03.费用定额
#### 新疆
- 建筑安装、市政工程费用定额（2020）
- 园林绿化工程费用定额（2014）

### 04.地区估价表
#### 新疆 → 阿克苏
- 房屋建筑与装饰工程单位估价表（2022）
- 安装工程单位估价表（2022）
- 市政工程单位估价表（2022）
- 园林绿化工程单位估价表（2014）

---

> **注意**：工程实施期间如遇新疆维吾尔自治区工程造价管理部门颁布新的建设工程计价标准、规则及消耗量定额时，依照新文件规定执行。
