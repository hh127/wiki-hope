# 新疆维吾尔自治区市政工程消耗量定额（2020）

## 基本信息

| 项目 | 内容 |
|------|------|
| 定额名称 | 新疆维吾尔自治区市政工程消耗量定额 |
| 发布年份 | 2020年 |
| 发布单位 | 新疆维吾尔自治区住房和城乡建设厅 |
| 适用范围 | 新疆维吾尔自治区市政工程 |

## 主要内容

本定额规定了新疆维吾尔自治区市政工程的人工、材料、机械台班消耗量标准。

### 适用范围

适用于新疆维吾尔自治区行政区域内新建、扩建、改建的市政工程。

### 主要章节

1. 土石方工程
2. 道路工程
3. 桥涵工程
4. 隧道工程
5. 给水工程
6. 排水工程
7. 燃气与集中供热工程
8. 路灯工程
9. 钢筋工程
10. 拆除工程
11. 措施项目
