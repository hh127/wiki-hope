# 新疆维吾尔自治区园林绿化工程消耗量定额（2014）

## 基本信息

| 项目 | 内容 |
|------|------|
| 定额名称 | 新疆维吾尔自治区园林绿化工程消耗量定额 |
| 发布年份 | 2014年 |
| 发布单位 | 新疆维吾尔自治区住房和城乡建设厅 |
| 适用范围 | 新疆维吾尔自治区园林绿化工程 |

## 主要内容

本定额规定了新疆维吾尔自治区园林绿化工程的人工、材料、机械台班消耗量标准。

### 适用范围

适用于新疆维吾尔自治区行政区域内新建、扩建、改建的园林绿化工程。

### 主要章节

1. 绿化工程
2. 园路、园桥工程
3. 园林景观工程
4. 假山工程
5. 水景工程
6. 园林设施工程
