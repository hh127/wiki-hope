# 新疆维吾尔自治区房屋建筑与装饰工程消耗量定额（2020）

## 基本信息

| 项目 | 内容 |
|------|------|
| 定额名称 | 新疆维吾尔自治区房屋建筑与装饰工程消耗量定额 |
| 发布年份 | 2020年 |
| 发布单位 | 新疆维吾尔自治区住房和城乡建设厅 |
| 适用范围 | 新疆维吾尔自治区房屋建筑与装饰工程 |

## 主要内容

本定额规定了新疆维吾尔自治区房屋建筑与装饰工程的人工、材料、机械台班消耗量标准。

### 适用范围

适用于新疆维吾尔自治区行政区域内新建、扩建、改建的工业与民用房屋建筑与装饰工程。

### 主要章节

1. 土石方工程
2. 地基处理与边坡支护工程
3. 桩基工程
4. 砌筑工程
5. 混凝土及钢筋混凝土工程
6. 金属结构工程
7. 木结构工程
8. 门窗工程
9. 屋面及防水工程
10. 保温、隔热、防腐工程
11. 楼地面装饰工程
12. 墙、柱面装饰与隔断、幕墙工程
13. 天棚工程
14. 油漆、涂料、裱糊工程
15. 其他装饰工程
16. 拆除工程
17. 措施项目
