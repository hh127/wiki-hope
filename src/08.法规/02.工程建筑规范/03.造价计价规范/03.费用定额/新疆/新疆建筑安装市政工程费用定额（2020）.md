# 新疆维吾尔自治区建筑安装、市政工程费用定额（2020）

## 基本信息

| 项目 | 内容 |
|------|------|
| 定额名称 | 新疆维吾尔自治区建筑安装、市政工程费用定额 |
| 发布年份 | 2020年 |
| 发布单位 | 新疆维吾尔自治区住房和城乡建设厅 |
| 适用范围 | 新疆维吾尔自治区建筑安装、市政工程 |

## 主要内容

本定额规定了新疆维吾尔自治区建筑安装、市政工程的费用计算标准和方法。

### 适用范围

适用于新疆维吾尔自治区行政区域内新建、扩建、改建的建筑安装、市政工程。

### 主要费用项目

1. **直接费**
   - 人工费
   - 材料费
   - 机械使用费

2. **间接费**
   - 规费
   - 企业管理费

3. **利润**

4. **税金**

### 工程类别划分

- 一类工程
- 二类工程
- 三类工程
- 四类工程
