# 房屋建筑与装饰工程工程量计算规范（GB50854-2013）

## 基本信息

| 项目 | 内容 |
|------|------|
| 标准编号 | GB50854-2013 |
| 标准名称 | 房屋建筑与装饰工程工程量计算规范 |
| 发布日期 | 2013年 |
| 实施日期 | 2013年 |
| 适用范围 | 房屋建筑与装饰工程工程量计算 |

## 主要内容

本规范规定了房屋建筑与装饰工程工程量计算的基本原则、计算方法和计算规则。

### 适用范围

适用于新建、扩建、改建的工业与民用房屋建筑与装饰工程的工程量计算。
