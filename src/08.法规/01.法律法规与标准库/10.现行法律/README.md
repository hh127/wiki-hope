# 现行法律

全国人民代表大会及其常务委员会制定的现行有效法律，按法律部门分类。

## 分类

- [宪法](10.宪法/) - 国家根本法
- [宪法相关法](20.宪法相关法/) - 国家机构、选举、组织法等
- [民法商法](30.民法商法/) - 民法典、公司法、合同法等
- [行政法](40.行政法/) - 行政管理、行政处罚、行政许可等
- [经济法](50.经济法/) - 财税、金融、市场监管等
- [社会法](60.社会法/) - 劳动、社会保障、特殊群体保护等
- [刑法](70.刑法/) - 刑法及刑事法律
- [诉讼与非诉讼程序法](80.诉讼与非诉讼程序法/) - 民事、刑事、行政诉讼法等
