# 法律法规与标准库

按法律效力级别分类整理的法律法规数据库。

## 分类

- [现行法律](10.现行法律/) - 全国人大及其常委会制定的法律
- [司法解释](20.司法解释/) - 最高人民法院、最高人民检察院司法解释
- [司法解释性质文件](30.司法解释性质文件/) - 司法解释性质的规范性文件
- [行政法规](40.行政法规/) - 国务院制定的行政法规
- [部门规章](50.部门规章/) - 国务院各部门制定的规章
- [废止法律](60.废止法律/) - 已被废止的法律法规
