# 造价分析与审核

## 造价分析内容

### 单方造价分析

```
单方造价 = 工程总造价 / 建筑面积
```

**常见单方造价参考（仅供参考）**：

| 结构类型 | 单方造价(元/m²) | 说明 |
|----------|-----------------|------|
| 砖混结构 | 1200-1800 | 多层住宅 |
| 框架结构 | 1500-2200 | 多层公建 |
| 框剪结构 | 1800-2800 | 高层住宅 |
| 剪力墙结构 | 2000-3200 | 高层住宅 |
| 钢结构 | 2500-4000 | 工业厂房 |

### 费用构成分析

```
人工费占比 = 人工费 / 分部分项工程费 × 100%
材料费占比 = 材料费 / 分部分项工程费 × 100%
机械费占比 = 机械费 / 分部分项工程费 × 100%
```

**参考比例**：
- 人工费：15%-25%
- 材料费：55%-70%
- 机械费：5%-15%

### 分部分项分析

按专业分析造价占比：
- 土建工程：60%-75%
- 装饰工程：15%-25%
- 安装工程：10%-20%

## 审核方法

### 全面审核法

逐项审核工程量、单价、费用：
- 优点：准确全面
- 缺点：工作量大

### 重点审核法

审核造价占比大的项目：
- 优点：效率高
- 缺点：可能遗漏

### 对比审核法

与类似工程对比分析：
- 优点：快速发现问题
- 缺点：需要数据库

### 分组计算审核法

将相关项目编为一组审核：
- 优点：逻辑清晰
- 缺点：需要经验

## 常见问题

| 问题类型 | 表现 | 处理方式 |
|----------|------|----------|
| 工程量多算 | 数量偏大 | 核对计算底稿 |
| 单价偏高 | 综合单价异常 | 市场询价对比 |
| 费率错误 | 取费标准错误 | 按规定调整 |
| 重复计价 | 项目重复 | 合并或删除 |
| 漏项 | 缺少项目 | 补充计价 |

## 审核要点

### 工程量审核

1. 核对计算规则是否正确
2. 检查扣减关系是否准确
3. 验证计算公式是否正确
4. 抽查重点项目的工程量

### 单价审核

1. 检查定额套用是否正确
2. 核对材料价格是否合理
3. 验证换算是否准确
4. 对比市场价是否偏差过大

### 费用审核

1. 检查费率标准是否正确
2. 核对计费基数是否准确
3. 验证税金计算是否正确
4. 检查不可竞争费是否按规定计取

## 成本控制建议

1. **设计阶段**：优化设计方案，控制含钢率、含混凝土量
2. **招标阶段**：合理编制招标控制价，避免过高或过低
3. **施工阶段**：严格变更签证管理，控制现场签证
4. **结算阶段**：认真审核结算资料，防止多计冒算
