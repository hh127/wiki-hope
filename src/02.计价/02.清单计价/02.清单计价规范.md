# 清单计价规范

## 计算依据

- 《建设工程工程量清单计价规范》GB 50500-2013

## 清单计价原则

1. **政府宏观调控**：统一项目编码、项目名称、计量单位、工程量计算规则
2. **企业自主报价**：综合单价由企业自主确定
3. **市场竞争形成价格**：通过招投标形成合同价

## 清单组成

### 分部分项工程量清单

| 序号 | 项目编码 | 项目名称 | 项目特征 | 计量单位 | 工程量 | 综合单价 | 合价 |
|------|----------|----------|----------|----------|--------|----------|------|
| 1 | 010101001 | 平整场地 | 土壤类别 | m² | xxx | xx | xx |

**项目编码结构**：
```
01 01 01 001
│  │  │  └── 清单项目顺序码
│  │  └───── 节顺序码
│  └──────── 章顺序码
└─────────── 附录顺序码（01-建筑、02-装饰、03-安装...）
```

### 措施项目清单

- 通用措施项目：安全文明施工、夜间施工、二次搬运等
- 专业措施项目：脚手架、模板、垂直运输等

### 其他项目清单

| 项目 | 说明 |
|------|------|
| 暂列金额 | 招标人确定，用于施工中可能发生的费用 |
| 暂估价 | 材料暂估价、专业工程暂估价 |
| 计日工 | 完成招标人提出的工程以外的零星工作 |
| 总承包服务费 | 总承包人为配合协调发包人进行的工程分包等 |

## 综合单价编制

### 综合单价构成

```
综合单价 = 人工费 + 材料费 + 机械使用费 + 管理费 + 利润 + 风险费
```

### 编制方法

1. **定额计价法**：套用预算定额，计算人材机费用，再取费
2. **成本测算法**：根据实际成本测算综合单价
3. **市场询价法**：向供应商询价确定单价

## 常见问题

| 问题 | 处理方式 |
|------|----------|
| 清单漏项 | 按合同约定处理，通常由投标人承担风险 |
| 工程量偏差 | 超过15%可调整综合单价 |
| 特征描述不符 | 按实际调整综合单价 |
| 暂估价确认 | 按实际确认价调整 |

## 实务要点

1. **项目特征**：必须准确描述，影响综合单价
2. **工程量**：由招标人提供，投标人复核
3. **综合单价**：包含全部费用，结算时一般不调整
