# 预算编制基础

## 编制依据

1. 施工图纸及设计说明
2. 工程量清单计价规范
3. 预算定额
4. 材料信息价
5. 施工组织设计
6. 招标文件及合同条款

## 工程造价构成

```
工程造价 = 分部分项工程费 + 措施项目费 + 其他项目费 + 规费 + 税金
```

### 分部分项工程费

```
分部分项工程费 = Σ(工程量 × 综合单价)
```

综合单价包含：
- 人工费
- 材料费
- 机械使用费
- 企业管理费
- 利润
- 一定范围内的风险费用

### 措施项目费

| 类别 | 项目 | 计算方式 |
|------|------|----------|
| 安全文明施工费 | 环境保护、文明施工、安全施工、临时设施 | 不可竞争费 |
| 夜间施工增加费 | 夜间施工 | 按规定计算 |
| 二次搬运费 | 材料二次搬运 | 按规定计算 |
| 冬雨季施工增加费 | 冬雨季施工 | 按规定计算 |
| 大型机械进出场及安拆费 | 大型机械 | 按定额计算 |
| 脚手架费 | 脚手架搭拆 | 按定额计算 |
| 模板及支架费 | 模板制作安装 | 按定额计算 |

### 其他项目费

- 暂列金额
- 暂估价
- 计日工
- 总承包服务费

### 规费

- 社会保险费（养老、失业、医疗、工伤、生育）
- 住房公积金
- 工程排污费

### 税金

- 增值税（一般计税9%，简易计税3%）

## 编制流程

```
1. 熟悉图纸
   ↓
2. 计算工程量
   ↓
3. 套用定额/组价
   ↓
4. 计算各项费用
   ↓
5. 汇总造价
   ↓
6. 编制说明
   ↓
7. 审核校对
```

## 注意事项

1. **工程量**：必须按计算规则准确计算，不能遗漏
2. **定额套用**：选择正确的定额子目，注意换算
3. **材料价格**：参考信息价，结合市场价
4. **费率**：按当地规定取费，注意费率调整
